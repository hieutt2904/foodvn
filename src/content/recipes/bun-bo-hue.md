  - { name: Xương bò, quantity: '1.5 kg' }
  - { name: Bắp bò, gân bò, quantity: '500 g + 300 g' }
  - { name: Chân giò heo, quantity: '500 g' }
  - { name: Giò heo (chả Huế), quantity: '300 g' }
  - { name: Sả cây, quantity: '10 cây' }
  - { name: Mắm ruốc Huế, quantity: '2 muỗng canh' }
  - { name: Ớt sa tế, dầu màu điều, quantity: '3 muỗng canh + 2 muỗng canh' }
  - { name: Bún bò sợi to, quantity: '1 kg' }
  - { name: Rau (bắp chuối bào, giá, rau muống chẻ, húng quế), quantity: '1 mâm' }

steps:
  - 'Chần xương bò và chân giò qua nước sôi, rửa sạch. Hầm xương với 6 lít nước trong 2 tiếng cùng sả đập dập (giữ lại 3 cây để phi).'
  - 'Cho bắp bò, gân bò, chân giò vào hầm cùng tới khi mềm, vớt ra ngâm nước lạnh rồi thái lát.'
  - 'Hoà tan mắm ruốc với 1 chén nước nóng, để lắng cặn, lấy phần nước trong cho vào nồi nước dùng.'
  - 'Phi thơm sả băm + tỏi với dầu màu điều, cho ớt sa tế vào tạo màu đỏ óng. Đổ vào nồi nước dùng.'
  - 'Nêm nước dùng với nước mắm, đường phèn, muối — vị phải đậm đà, cay nồng, chua nhẹ từ mắm ruốc.'
  - 'Chả Huế hấp chín, thái khoanh.'
  - 'Trần bún vào tô, xếp bắp bò, gân, giò heo, chả Huế. Chan nước dùng đang sôi. Ăn kèm rau sống và chanh ớt.'
tips: |
  Mắm ruốc là gia vị KHÔNG THỂ THIẾU của bún bò Huế. Phải để lắng cặn và chỉ dùng phần nước trong, nếu không nước dùng sẽ đục và đắng. Sa tế tự làm từ ớt khô + dầu sẽ thơm hơn loại đóng hộp.
---

Bún bò Huế nổi tiếng với hương vị cay nồng đậm đà, là tô bún "định danh" cho ẩm thực miền Trung. Một tô bún ngon phải có đủ 4 vị: cay, ngọt, mặn, chua tinh tế.

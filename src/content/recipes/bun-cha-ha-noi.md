title: Bún chả Hà Nội
description: Thịt nướng than hoa thơm lừng ăn kèm bún tươi, nước chấm chua ngọt và rau sống tươi mát — đặc sản phố cổ Hà Nội.
region: bac
meal: trua
mainDish: true
difficulty: trung-binh
prepTime: 40
cookTime: 25
servings: 4
featured: true
tags: [bun, thit-nuong, ha-noi]
ingredients:
  - { name: Thịt ba chỉ, quantity: '400 g' }
  - { name: Thịt nạc vai (để viên), quantity: '300 g' }
  - { name: Bún tươi, quantity: '800 g' }
  - { name: Hành khô, tỏi, quantity: '5 củ + 1 củ tỏi' }
  - { name: Nước mắm, đường, mật ong, dầu hào, quantity: '2-3-1-1 muỗng canh' }
  - { name: Hạt tiêu, dầu màu điều, quantity: 'vừa đủ' }
  - { name: Đu đủ xanh, cà rốt (làm dưa góp), quantity: '200 g + 100 g' }
  - { name: Giấm, đường, tỏi, ớt, chanh (nước chấm), quantity: 'vừa đủ' }
  - { name: Rau sống (xà lách, kinh giới, tía tô, húng quế), quantity: '1 mớ' }
steps:
  - 'Thịt ba chỉ thái lát mỏng 0.5cm. Thịt nạc vai băm nhuyễn rồi viên dẹt.'
  - 'Ướp thịt với hành khô + tỏi băm + nước mắm + đường + mật ong + dầu hào + tiêu + dầu màu điều, để 30 phút cho thấm.'
  - 'Đu đủ và cà rốt thái lát mỏng, bóp muối 10 phút rồi rửa sạch, vắt khô. Pha giấm đường với tỷ lệ 2:1, đổ vào ngâm 15 phút làm dưa góp.'
  - 'Pha nước chấm: 4 phần nước ấm + 1 phần nước mắm + 1 phần đường + 1 phần giấm + nước cốt 1/2 quả chanh. Cho tỏi ớt băm và dưa góp vào.'
  - 'Nướng thịt trên than hoa (hoặc bếp than/chảo gang) đến khi cháy cạnh, mặt thịt vàng nâu thơm lừng.'
  - 'Thả thịt nướng vào bát nước chấm còn ấm, để thịt ngấm nước chấm.'
  - 'Dọn bún ra đĩa, kèm rau sống và bát nước chấm có thịt. Chấm bún vào nước chấm khi ăn.'
tips: |
  Bí quyết là phải nướng bằng than hoa thực sự để có khói thơm đặc trưng. Nếu dùng chảo, hãy thêm chút dầu màu điều và nướng lửa lớn. Nước chấm phải ấm để thịt nướng giữ được độ mềm.
---

Bún chả nổi tiếng đến mức cựu Tổng thống Mỹ Obama đã ghé Hà Nội thưởng thức. Đây là món gắn liền với những trưa hè phố cổ, mùi than hoa thơm lừng từng ngõ nhỏ.

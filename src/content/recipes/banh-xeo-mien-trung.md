region: trung
meal: an-vat
mainDish: false
difficulty: trung-binh
prepTime: 30
cookTime: 30
servings: 4
tags: [banh, ran, an-vat]
ingredients:
  - { name: Bột bánh xèo (hoặc bột gạo + bột chiên giòn), quantity: '300 g' }
  - { name: Nước cốt dừa, quantity: '200 ml' }
  - { name: Bột nghệ, quantity: '1 muỗng cà phê' }
  - { name: Tôm sú, thịt ba chỉ, quantity: '200 g + 200 g' }
  - { name: Giá đỗ, hành lá, quantity: '300 g + 1 mớ' }
  - { name: Bánh tráng cuốn, rau sống (xà lách, cải xanh, húng quế, dưa leo), quantity: 'vừa đủ' }
  - { name: Đậu phộng rang, quantity: '50 g' }
  - { name: Nước mắm chua ngọt, quantity: 'vừa đủ' }
steps:
  - 'Pha bột: trộn bột bánh xèo + nước cốt dừa + 300ml nước + bột nghệ + chút muối. Khuấy đều, để nghỉ 30 phút.'
  - 'Tôm rửa sạch để nguyên vỏ hoặc bóc tuỳ thích. Thịt ba chỉ thái lát mỏng. Ướp cả hai với chút muối, tiêu, hành tím băm.'
  - 'Hành lá thái nhỏ, cho vào tô bột.'
  - 'Bắc chảo gang nhỏ (đường kính 16-18cm) lên bếp, làm nóng già với chút dầu.'
  - 'Cho 1-2 con tôm + vài lát thịt vào chảo xào săn. Đổ 1 muôi bột láng đều mặt chảo, nghiêng nhanh để bột phủ kín đáy thành lớp MỎNG.'
  - 'Rắc nắm giá đỗ lên một nửa bánh, đậy nắp 1 phút cho giá chín. Mở nắp, chiên thêm cho bánh giòn rụm.'
  - 'Gập đôi bánh, dọn ra đĩa. Pha nước mắm chua ngọt với tỏi ớt băm, đậu phộng giã.'
  - 'Ăn cuốn: bánh tráng + rau sống + miếng bánh xèo + chấm nước mắm.'
tips: |
  Bánh xèo miền Trung NHỎ và GIÒN hơn bánh xèo miền Nam. Bí quyết giòn: chảo phải nóng già, bột phải mỏng và dầu đủ nhiều. Không dùng nước cốt dừa quá nhiều sẽ bị béo ngậy không đặc trưng.
---

Bánh xèo miền Trung thường nhỏ hơn bánh xèo miền Nam, kích thước vừa lòng bàn tay nhưng giòn rụm. Đây là món ăn vặt yêu thích ở các tỉnh từ Bình Định đến Quảng Trị.

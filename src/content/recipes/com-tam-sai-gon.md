ingredients:
  - { name: Gạo tấm, quantity: '600 g' }
  - { name: Sườn cốt lết heo, quantity: '4 miếng (~800 g)' }
  - { name: Trứng gà, thịt heo xay, mộc nhĩ (chả trứng), quantity: '4 + 200 g + 30 g' }
  - { name: Bì heo (mua sẵn) hoặc da heo + thịt thăn + thính, quantity: '200 g' }
  - { name: Hành tím, tỏi, sả băm, quantity: '5 củ + 1 củ + 2 cây' }
  - { name: Mật ong, dầu hào, nước mắm, đường, quantity: '1-1-2-2 muỗng canh' }
  - { name: Đồ chua (cà rốt + củ cải), dưa leo, cà chua, quantity: 'vừa đủ' }
  - { name: Nước mắm chua ngọt (chấm), mỡ hành, quantity: 'vừa đủ' }
steps:
  - 'Vo gạo tấm nhẹ tay, nấu với tỷ lệ nước 1:1.2 (ít hơn cơm thường).'
  - 'Ướp sườn: hành tỏi sả băm + mật ong + dầu hào + nước mắm + đường + dầu màu điều + tiêu. Ướp tối thiểu 2 tiếng (qua đêm càng ngon).'
  - 'Nướng sườn trên than hoa hoặc lò 200°C trong 15-20 phút, phết nước ướp lên 2 lần.'
  - 'Làm chả trứng: trộn thịt xay + trứng + mộc nhĩ ngâm băm + hành tím + nước mắm + tiêu. Đổ vào khuôn, hấp 25 phút. Quét thêm lòng đỏ trứng lên mặt và nướng vàng.'
  - 'Mỡ hành: hành lá thái nhỏ, đổ dầu nóng vào trộn.'
  - 'Pha nước mắm chua ngọt: 3 nước + 2 đường + 2 nước mắm + 1 nước cốt chanh + tỏi ớt.'
  - 'Dọn đĩa: cơm tấm, sườn nướng, chả trứng cắt miếng, bì, đồ chua, dưa leo, cà chua, mỡ hành. Rưới nước mắm.'
tips: |
  Cơm tấm CHUẨN dùng gạo tấm (gạo gãy) chứ không phải gạo thường. Sườn ngon phải ướp đủ thời gian và nướng than hoa. Mỡ hành là chi tiết nhỏ nhưng tạo nên linh hồn của đĩa cơm.
---

Cơm tấm là món "đặc sản" Sài Gòn theo nghĩa đúng nhất — sinh ra từ tinh thần tiết kiệm của người nghèo (dùng gạo gãy không bán được), nay trở thành món ăn yêu thích từ vỉa hè đến nhà hàng cao cấp.

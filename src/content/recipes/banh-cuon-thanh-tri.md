ingredients:
  - { name: Bột gạo tẻ, quantity: '200 g' }
  - { name: Bột năng, quantity: '50 g' }
  - { name: Nước lọc, quantity: '700 ml' }
  - { name: Thịt heo xay, quantity: '200 g' }
  - { name: Mộc nhĩ, nấm hương, quantity: '30 g + 5 cái' }
  - { name: Hành tím băm, quantity: '3 củ' }
  - { name: Chả lụa (giò lụa), quantity: '200 g' }
  - { name: Hành phi, rau thơm, quantity: 'vừa đủ' }
  - { name: Nước mắm, đường, giấm, chanh, ớt (chấm), quantity: 'vừa đủ' }
steps:
  - 'Pha bột: trộn bột gạo + bột năng + nước + chút muối + 1 muỗng canh dầu ăn. Để bột nghỉ ít nhất 2 tiếng (hoặc qua đêm) để bánh dai.'
  - 'Ngâm mộc nhĩ và nấm hương vào nước ấm 20 phút, băm nhỏ.'
  - 'Phi thơm hành tím, cho thịt xay vào xào tới chín. Cho mộc nhĩ, nấm hương vào, nêm nước mắm, tiêu, đảo đều.'
  - 'Bắc chảo chống dính trên bếp, lửa vừa. Múc một muôi bột láng đều đáy chảo, đậy nắp 30 giây cho bánh chín.'
  - 'Lấy bánh ra mâm phết dầu, cho nhân vào giữa, cuộn lại. Rắc hành phi lên trên.'
  - 'Pha nước chấm: 4 phần nước ấm + 1 phần nước mắm + 1 phần đường + nước cốt chanh, thêm ớt thái lát.'
  - 'Dọn bánh cuốn ra đĩa với chả lụa thái lát, rau thơm (rau mùi, kinh giới) và bát nước chấm.'
tips: |
  Bánh cuốn ngon là phải MỎNG và TRONG. Bột phải được nghỉ đủ giờ. Khi tráng, chảo phải đủ nóng nhưng không nóng quá. Nước chấm bánh cuốn pha LOÃNG hơn nước chấm bún chả.
---

Bánh cuốn Thanh Trì là đặc sản làng Thanh Trì (Hà Nội), nổi tiếng nhờ kỹ thuật tráng bánh điêu luyện cho ra những lớp bánh mỏng như giấy lụa.

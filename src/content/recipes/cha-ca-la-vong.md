---
title: Chả cá Lã Vọng
description: Cá lăng tẩm nghệ mắm tôm, nướng vàng rồi xào với thì là và hành lá — đặc sản trăm năm của Hà Nội.
region: bac
meal: toi
mainDish: true
difficulty: trung-binh
prepTime: 60
cookTime: 20
servings: 4
tags: [ca, mam-tom, ha-noi]
ingredients:
  - { name: Cá lăng (hoặc cá quả) phi-lê, quantity: '600 g' }
  - { name: Bột nghệ, riềng băm, quantity: '1 muỗng cà phê + 2 muỗng canh' }
  - { name: Mẻ, mắm tôm, quantity: '2 muỗng + 1 muỗng canh' }
  - { name: Thì là, hành lá, quantity: '1 mớ to mỗi loại' }
  - { name: Bún tươi, quantity: '500 g' }
  - { name: Lạc rang, bánh đa nướng, quantity: '100 g + 4 cái' }
  - { name: Rau sống (húng láng, mùi tàu), quantity: '1 mớ' }
  - { name: Mắm tôm pha (chấm), quantity: 'vừa đủ' }
steps:
  - 'Phi-lê cá thái miếng vuông 3x3cm. Ướp với riềng băm + bột nghệ + mẻ + mắm tôm + chút dầu ăn, để tủ lạnh ít nhất 1 tiếng.'
  - 'Thì là, hành lá cắt khúc 4-5cm.'
  - 'Nướng cá trên than hoa hoặc lò nướng 200°C trong 10 phút đến khi cá vàng đều hai mặt (có thể bỏ qua bước này, xào trực tiếp).'
  - 'Bắc chảo lớn lên bếp, cho nhiều dầu ăn nóng. Cho cá vào áp chảo vàng giòn hai mặt.'
  - 'Khi cá gần chín, cho thì là và hành lá vào đảo nhanh tay, tắt bếp ngay khi rau chín tới.'
  - 'Pha mắm tôm: mắm tôm + đường + nước cốt chanh + rượu trắng + ớt, đánh bông tới nổi bọt.'
  - 'Dọn cá ra chảo gang nóng (để giữ nhiệt), ăn kèm bún, lạc rang, bánh đa, rau sống và mắm tôm pha.'
tips: |
  Mẻ là gia vị đặc trưng tạo độ chua thanh cho món chả cá. Nếu không có mẻ, có thể thay bằng sữa chua không đường. Khi xào, lửa phải lớn để cá vàng giòn mà không bị bở.
---

Chả cá Lã Vọng là một trong những món ăn nổi tiếng nhất Hà Nội, gắn liền với nhà hàng cùng tên ở phố Chả Cá đã có hơn 130 năm tuổi.
